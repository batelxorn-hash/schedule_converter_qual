# Schedule Converter

Програмний засіб для конвертації академічного розкладу занять у формат iCalendar (.ics).

## Функціональні можливості

- імпорт розкладу з TXT;
- імпорт розкладу з CSV;
- імпорт розкладу з XLSX;
- імпорт розкладу з PDF;
- генерація файлів формату ICS;
- перевірка конфліктів занять;
- підтримка конфігураційних JSON-шаблонів;
- графічний інтерфейс користувача на Tkinter.

## Використані технології

- Python 3.10
- Tkinter
- iCalendar
- Pandas
- PDFPlumber
- JSON

## Запуск

pip install -r requirements.txt

python main.py
